# Reform Flow

リフォーム案件の進捗、工程、売上見込み、入金予定をスマホ中心に管理する個人用アプリです。

## 公開URL

GitHub Pagesのデプロイ後、次のURLで開けます。

https://stakizawa250-cmyk.github.io/reform-flow/

## 初回だけ必要な設定

このリポジトリではアプリ本体とGitHub Pages用の自動公開ワークフローを用意済みです。
GitHub側の権限仕様により、初回だけリポジトリ設定でPagesを有効化してください。

1. GitHubで `Settings` を開く
2. 左メニューの `Pages` を開く
3. `Build and deployment` の `Source` を `GitHub Actions` にする
4. 保存後、`Actions` タブから `Deploy static app to GitHub Pages` を再実行する

再実行が成功すると、上記の公開URLでアプリを開けます。
